## Cómo ejecutar el hook de epoch (operación)

1. Copiar y completar mapa de wallets:
   - `goalworld_oracle/contributor_wallet_map.example.json` → `contributor_wallet_map.json`

2. Variables de entorno mínimas:

```bash
export ORACLE_KEYPAIR_PATH=~/.config/solana/id.json
export TREASURY_ATA=<pubkey_ata_tesoreria>
export TOKEN_MINT=<pubkey_mint_gch>
export CONTRIBUTOR_EPOCH_POOL=700000000000   # lamports (ej. 700k GCH con 6 dec)
export RPC_URL=http://127.0.0.1:8899
```

3. Ejecutar:

```bash
cd goalworld_oracle
npm run contributor-epoch-hook
```
