# VISION_AND_AI_GOVERNANCE

Este documento consolida la directiva de gobernanza del protocolo:

- Ver visión inspiradora: `docs/VISION_AND_AI_GOVERNANCE_INSPIRADORA.md`
- Ver instrucción técnica obligatoria: `docs/VISION_AND_AI_GOVERNANCE_TECNICA.md`

## Reglas activas del repositorio

1. Founder capture hard-cap: **1% máximo**.
2. Fondo de constructores: **10%** del valor generado, distribuido con métricas verificables.
   - Este 10% también financia **toda la capa de APIs/modelos/infra de desarrollo**, incluyendo marketing y growth.
   - No se crea un bucket separado de marketing fuera del Builder Fund.
3. No se aceptan nuevas rutas de extracción cerrada (fees, royalties, treasury control) por encima del límite.
4. Cualquier cambio económico debe documentar impacto en:
   - emisión
   - quema
   - circulación
   - fairness para usuarios y contributors.

## Prioridad de ejecución

1. Fase 0: auditoría + documentación de parámetros actuales.
2. Fase 1: reducción efectiva de capture >1%.
3. Fase 2+: fondo de constructores, hooks para CEO Agent y gobernanza progresiva.

Este archivo es una referencia canónica para implementación y revisión.
