# ⚽ goalworld: Instrucciones de Creación de Imágenes de Jugadores (Capa L2)

Este documento contiene las especificaciones técnicas y el diccionario maestro para generar los activos individuales de los jugadores (Capa L2).

## 📐 Especificaciones de Activos (L2 Layer)
1.  **Dimensiones:** 2000 x 3000 px (Relación 2:3).
2.  **Fondo:** Blanco Puro o Verde Croma sólido.
3.  **Iluminación:** Rim lights en **Verde Solana (#14f195)** y **Púrpura Solana (#9945ff)**.
4.  **Escala:** Jugador centrado (70-75% del alto).
