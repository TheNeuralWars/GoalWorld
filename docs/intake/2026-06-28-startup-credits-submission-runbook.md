**Time:** ~10 min | **Login:** AWS Builder ID | **URL:** https://aws.amazon.com/activate/apply/

### Steps

2. Sign in with AWS Builder ID (create if needed)
3. Fill profile completely:
   - Company name: `GoalWorld`
   - Website: `https://goalworld.fun`

   - Stage: `Early stage / Pre-revenue`
   - Employees: `1-10`
   - Funding: `Self-funded / bootstrapped`

### Key answers to copy

## TROUBLESHOOTING

**"My company doesn't have a legal entity yet"**
→ AWS, Google, and Microsoft all accept pre-incorporation startups. Use "GoalWorld" as company name. For legal entity fields, select "Not yet incorporated" or leave blank if possible.

**"They ask for an EIN / tax ID"**
→ Leave blank or enter "N/A — pre-incorporation". These programs accept international and pre-entity startups.

**"They ask for funding documents"**
→ State "self-funded / bootstrapped". No documents needed for bootstrapped stage.

**"Application asks for a referral code"**
→ Leave blank. Not required for any of the three programs.
